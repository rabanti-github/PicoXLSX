Exceptions

StyleExceptionThrows a StyleException if the default style was malformed or if the active style cannot be referenced
Remarks
Recognized are the following data types: string, int, double, float, long, DateTime, TimeSpan, bool. All other types will be casted into a string using the default ToString() method.
If the cell object already has a style definition, and a style or active style is defined, the cell style will be merged, otherwise just set
